Saya membuat frontend menggunakan React, TypeScript, Tailwind CSS, Axios, dan React Router. 
Implementasi dilakukan dengan pendekatan component-based sehingga komponen dapat digunakan kembali dan mudah dipelihara. 
Frontend terintegrasi dengan REST API Perpustakaan Golang menggunakan Axios Service Layer. 
Fitur yang diimplementasikan meliputi autentikasi JWT, dashboard, dan manajemen data buku. 
Struktur project dipisahkan berdasarkan pages, components, services, dan API layer agar lebih scalable dan mudah dikembangkan. 
Tampilan dibuat responsive menggunakan Tailwind CSS sehingga dapat digunakan pada desktop maupun perangkat mobile. 
Endpoint yang digunakan antara lain login (POST /login) dan daftar buku (GET /api/buku).
